# MarkGate株式会社 コーポレートサイト

ハイクラス求職者専用の「品質保証型」マッチングプラットフォーム事業 **MarkGate（マークゲート）** を
運営する MarkGate株式会社 のコーポレートサイトです。
**同一コンテンツ・3つのデザイン案** を用意し、比較・選択できる構成にしています。

> **トップアドバイザーだけが、開ける扉。**
> スカウトの「量の競争」で壊れた転職市場を、求職者が実績でCAを“指名”する「質の競争」へ。

## デザイン3案

トップページ（`index.html`）は **デザイン選択ギャラリー** です。各案のライブプレビューを表示し、クリックで全画面の各パターンを開けます。

| # | 名称 | 配色 | 書体 | 方向性 | ファイル |
|---|------|------|------|--------|----------|
| 01 | **Editorial Luxury（品格と信頼）** | ネイビー × ゴールド | 明朝（Shippori Mincho） | 会員制クラブ的・高級・編集的。ハイクラス感を最も強く表現 | `patterns/01-luxury.html` |
| 02 | **Modern Trust（明快な信頼）** | 白 × インディゴ | ゴシック太字（Noto Sans JP / Plus Jakarta） | クリーンなSaaS調。比較表・ステップ図で仕組みを明快に説明 | `patterns/02-clarity.html` |
| 03 | **Bold Disruptor（先進・ダーク）** | 漆黒 × エメラルド | 大型ディスプレイ（Space Grotesk） | 大胆なダークテック。挑戦的・先進的なブランド姿勢 | `patterns/03-bold.html` |

3案は **配色・書体・レイアウト・動き** のみが異なり、掲載コンテンツは共通です。

## 構成

```
.
├── index.html                 # デザイン選択ギャラリー（トップ）
├── patterns/
│   ├── 01-luxury.html         # 案01 Editorial Luxury（1ファイル完結）
│   ├── 02-clarity.html        # 案02 Modern Trust（1ファイル完結）
│   └── 03-bold.html           # 案03 Bold Disruptor（1ファイル完結）
├── assets/
│   ├── css/style.css          # 旧・現行サイト用スタイル（previews から参照）
│   ├── js/main.js             # 旧・現行サイト用スクリプト
│   └── img/favicon.svg        # ゲートをモチーフにしたロゴ／ファビコン
├── previews/                  # 初期ラフ案（5パターン）ギャラリー（参考用）
├── vercel.json                # 静的ホスティング設定（Vercel）
└── README.md
```

各パターンは **1ファイル完結（CSS/JSインライン）** なので、単体でそのまま共有・デプロイできます。

### 共通の掲載セクション

1. ヒーロー（タグライン／2種CTA）
2. 課題 — 「量の競争」で壊れた転職市場（同一DBへの殺到・受信箱の飽和・数%の返信率・担当者ガチャ・多重登録）
3. コンセプト — 「質の競争（指名型）」への転換（量 vs 質の対比）
4. 仕組み — 品質を“保証”する4つの仕組み（入会審査／継続モニタリング／ハイクラス限定の好循環／双方向×上限付きスカウト）
5. 3つの強み
6. 求職者の方へ
7. アドバイザー（CA）の方へ
8. ご利用の流れ（4ステップ）
9. 数値で見るMarkGate（コンセプト値）
10. 会社概要
11. お問い合わせフォーム

## ローカルでの確認

ビルド不要の静的サイトです。任意のHTTPサーバで開けます（トップのプレビューは iframe を使うため、`file://` 直開きより HTTP サーバ推奨）。

```bash
# 例：Python
python3 -m http.server 8000
# → http://localhost:8000

# 例：Node
npx serve .
```

## デプロイ

`vercel.json` を同梱しているため、Vercel にそのまま静的サイトとしてデプロイできます。
（GitHub Pages / Netlify / S3 など任意の静的ホスティングでも動作します）
